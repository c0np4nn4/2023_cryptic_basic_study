## 규칙
- 각 스터디원은 각각의 스터디 폴더 내에 <u>***자신의 닉네임으로 된 폴더***</u> 만 관리합니다.
  - 예) `c0np4nn4` 는 <kbd>DES/c0np4nn4/</kbd> 폴더 내용만 수정합니다.
- Commit 메세지에 제한은 없습니다.
- 만약 참고한 코드가 있다면 **출처를 명시**할 것을 권장합니다.
